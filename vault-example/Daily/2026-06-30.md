### 今日学习
- MCP Server 是 AI 与外部工具或数据源之间的标准接口；当前 ResearchGPT 负责安全地读写 Obsidian Vault。
- `npm install` 用于安装 Node.js 项目依赖，`npm install -g` 表示全局安装命令行工具。
- `127.0.0.1` 只代表本机；ChatGPT 云端无法直接访问，因此需要 ngrok 提供公网 HTTPS 隧道。
- ngrok authtoken 与 `MCP_ACCESS_TOKEN` 是两种不同凭据：前者用于登录 ngrok，后者用于保护 MCP Server。
- Git、curl 等命令行工具不会自动继承所有应用的代理设置，需要单独配置代理或环境变量。

### 开发
- MCP Server 已实现 Token 鉴权、Vault 路径隔离、敏感目录阻断、绝对路径与 `..` 越界防护、符号链接/Junction 逃逸防护。
- 当前支持的 MCP 工具仅覆盖文件浏览、读取、创建/覆盖和追加；尚不支持全文搜索、重命名、移动、删除、Frontmatter 更新和语义检索。
